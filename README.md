# Jujutsu Battle Bot 🤖⚔️ v2.0

Telegram-бот с карточной игрой по мотивам аниме "Магическая битва" (Jujutsu Kaisen).

## 🎮 Возможности

### Основные функции:
- 🎴 **Карточная система** - 50+ карт персонажей, оружия, шикигами, артефактов и пактов
- 👹 **PvE Арена** - 20+ проклятий разной сложности
- ⚔️ **PvP Бои** - бросай вызов другим игрокам
- 📖 **Сюжетная Кампания** - 7 сезонов, 25+ уровней, финальный босс Сукуна
- 📅 **Ежедневные Награды** - награды за вход каждый день с нарастающими бонусами
- 📋 **Ежедневные Задания** - выполняй квесты для получения наград
- 🏆 **Достижения** - 30+ достижений с наградами
- 👑 **Титулы** - экипируй титулы для бонусов к характеристикам
- 🏫 **Техникум** - обучайся техникам и способностям
- ✨ **Техники** - 30+ техник включая врожденные, территории, способности
- 🏪 **Рынок** - покупай, продавай и обменивай карты
- 🔄 **Обмен** - меняйся картами с друзьями
- 👥 **Друзья** - добавляй друзей и сражайся с ними

### Боевая Система:
- 💧 **Проклятая Энергия (CE)** - каждая карта имеет свою CE
- ⚡ **Способности** - Красный, Фиолетовый, Бугорок и др.
- 🏛️ **Территории** - создавай свою территорию и сражайся в битвах территорий
- 🛡️ **Простая Территория** - защищайся от чужих территорий
- ⚫ **Черная Молния** - критические удары с шансом срабатывания
- 🎲 **4 Слота** - персонаж, 2x оружие/шикигами, врожденная техника

### Прогрессия:
- ⭐ **Уровни** - получай опыт за бои, сообщения и задания
- 💎 **Очки** - трать на улучшение карт
- 🪙 **Монеты** - используй для техникума и рынка
- ⬆️ **Прокачка** - улучшай атаку, защиту, скорость, HP и CE

### Уровни Сложности:
- 🟢 **Легкий** - 0.5x награды, нет штрафов
- 🔵 **Нормальный** - 1x награды (стандарт)
- 🟠 **Сложный** - 1.5x награды, сильные враги
- 🔴 **Хардкор** - 2x награды, смерть = конец персонажа

## 🛠️ Установка

### 1. Клонируй репозиторий:
```bash
git clone <repo-url>
cd jujutsu_bot
```

### 2. Создай виртуальное окружение:
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. Установи зависимости:
```bash
pip install -r requirements.txt
```

### 4. Настрой конфигурацию:
```bash
cp .env.example .env
# Отредактируй .env и добавь свой BOT_TOKEN
```

Получить токен можно у [@BotFather](https://t.me/BotFather)

### 5. Запусти бота:
```bash
python bot.py
```

## 📜 Команды

### Основные:
- `/start` - Начать игру, получить стартовую карту
- `/profile` - Посмотреть свой профиль
- `/inventory` - Открыть инвентарь с картами
- `/battle` - Меню боев
- `/daily` - Ежедневные награды и задания
- `/campaign` - Сюжетная кампания
- `/academy` - Техникум магии
- `/market` - Рынок карт
- `/achievements` - Достижения и титулы
- `/difficulty` - Изменить уровень сложности

### Социальные:
- `/addfriend @username` - Добавить друга
- `/promo КОД` - Использовать промокод

### Админские:
- `/admin` - Админ панель
- `/givecard @username Карта [уровень]` - Выдать карту
- `/givetech @username Техника` - Выдать технику
- `/givecurrency @username ТИП КОЛИЧЕСТВО` - Выдать валюту
- `/createpromo КОД ОПИСАНИЕ [параметры]` - Создать промокод
- `/setadmin @username` - Назначить админа
- `/broadcast ТЕКСТ` - Рассылка всем игрокам

В файле `config.py` можно настроить:

```python

PVE_COOLDOWN = 60          # Кулдаун PvE (сек)

```

### Карты:

- 🔵 **Rare** (Редкая) - 30%
- 🟣 **Epic** (Эпическая) - 15%
- 🟡 **Legendary** (Легендарная) - 4.5%

### Техники:
- 🟢 **Common** - базовые техники
- 🔵 **Rare** - редкие способности
- 🟣 **Epic** - эпические техники
- 🟡 **Legendary** - легендарные (6 глаз, бесконечность)
- 🔴 **Mythical** - мифические (Сукуна)

## 🎴 Система Слотов

### 4 Слота в колоде:
1. **Слот 1** - Главный персонаж (обязательно)
2. **Слот 2** - Оружие / Шикигами / Пакт
3. **Слот 3** - Оружие / Шикигами / Пакт
4. **Слот 4** - Врожденная техника (6 глаз, бесконечность и т.д.)

## ⚔️ Боевая Система

### Порядок боя:
1. Определяется кто первый (по скорости)
2. Игроки по очереди используют:
   - Обычная атака
   - Способности (тратят CE)
   - Территория
   - Защитная стойка
3. Черная молния может сработать случайно
4. Побеждает тот, кто оставил HP врага = 0

### Проклятая Энергия:
- Каждая карта имеет свою CE
- CE восстанавливается каждый ход
- Способности тратят CE
- Шесть Глаз уменьшает стоимость CE на 50%

## 🏫 Техникум
